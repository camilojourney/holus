# Phase 5: Intent Audit

Check if implementation solves the user problem, not just the literal spec.

## Focus Areas

- intent alignment
- UX clarity and flow
- action labels and feedback quality
- accessibility basics
- confusion/friction points

## Output

- intent alignment table
- UX issues with minimal fix suggestions
- verdict: SHIPS WELL / FUNCTIONAL BUT MISALIGNED / MISSES INTENT

## Prompt Template

```text
Evaluate whether this implementation matches the user's intent and UX expectations.
Highlight mismatches and suggest minimal fixes.
```
