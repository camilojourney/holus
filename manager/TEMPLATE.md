# Manager Log Template

## Daily Checks (3x daily)
1. **Morning (varies by repo)**: Full health check
2. **Midday**: Quick status + any new issues  
3. **Evening**: Daily metrics + priorities for tomorrow

## Health Check Sections
- Pipeline/System status: ✅/⚠️/❌
- Worker status: All green? Any failures?
- Issues found: [description] → [fix applied] → [verified]
- Tasks created: What future agents should work on

## Format
- Consistent naming: manager/YYYY-MM-DD.md
- Daily updates per schedule
- Context for Manager of Managers
- Clear action items

See job-tracker/manager/ for detailed example.
