# Holus Context

## Product Summary

Personal AI Agent Workforce for local-first automation.

## Vision

Deploy specialized AI agents on local machine for 24/7 personal assistance.

## Operating Constraints

- Follow repository-native tooling and deployment conventions.
- Keep changes backward-compatible unless a migration is explicitly planned.
- Capture high-impact decisions in docs/specs before implementation.

## Core Commands

- Install: `pip install -r requirements.txt`
- Dev: `python -m holus.main`
- Test: `python -m pytest tests/`
- Lint: `flake8 core/ agents/ tools/`
- Build: `No build step (runtime Python app)`
